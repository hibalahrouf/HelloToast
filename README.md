# HelloToast App

Cette application Android simple a été développée avec Java sur Android Studio.

##  Fonctionnalités
- Affichage d’un compteur initialisé à 0  
- Incrémentation du compteur avec un bouton  
- Affichage d’un message (Toast) lors du clic  

## Technologies utilisées
- Java  
- Android SDK (API 24+)  
- XML pour l’interface utilisateur  

##  Exécution
1. Ouvrir le projet dans Android Studio  
2. Compiler le projet  
3. Lancer sur un émulateur ou un appareil Android
